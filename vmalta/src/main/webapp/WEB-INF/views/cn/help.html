帮助

VisitMalta的门户信息设计符合导航指引的要求。

通过Microsoft Internet Explorer 7浏览器或更高版本可以访问所有页面。下面的信息将帮助您了解我们的网站适用什么样的标准以及如何处理特定的文件类型。

导航

VisitMalta每个网页顶部均为导航栏。菜单中每个选项都有一个二级下拉式菜单，帮助您进入第二层次的页面。在页面的底部可以在找到另外一种非图形化导航。底部菜单用于显示主要选项的子选项。

网站说明

VisitMalta门户网站在设计时已考虑到所有用户。标准网页采用1024×768的256色分辨率。

浏览器兼容性

VisitMalta门户网站上的浏览器包括微软的Internet Explorer 7及以上版本、Mozilla Firefox和Chrome，已经全部经过测试。VisitMalta门户使用层叠样式表（CSS）对每个页面中的文本信息进行格式化。

页面标记

如需标记用户选定的某个特定页面，请打开浏览器的菜单选项，将该页面添加至收藏夹。

搜索

用户必须在文本搜索框内输入一个单词或一组单词，然后按回车键或点击Go按钮，符合搜索条件的对应文件清单将呈现在用户面前。这些结果将会分类显示，匹配度最高的结果将会优先显示，然后按关联性下降依次显示。

提交反馈

VisitMalta门户在“联系我们”选项中提供了客户服务信息建议意见反馈表。填妥表格后，用户必须点击页面底部的提交按钮，反馈将会发送到马耳他旅游局。我们不能对每一项查询都作出回复，在必要时，我们会读取信息并采取相应的行动。

保存网页

您可以将浏览器显示的任何页面保存到本地驱动器。请选择文件，然后将其另存为浏览器中的菜单选项。

信息

VisitMalta门户内有一些网页没有采用HTML或ASP.net等格式。这些都是可转化的文档文件（PDF格式）。需要读取这些文件的用户，可以从互联网上免费下载Adobe Acrobat Reader软件。

Javascript

 

本网站充分利用JavaScript的功能。请确保在浏览器设置中启用JavaScript功能，以获得最佳体验。
